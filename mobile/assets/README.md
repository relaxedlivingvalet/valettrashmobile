# Relaxed Living Valet Branding Assets

This folder contains branding assets for the Relaxed Living Valet Flutter application.

## Asset Structure

### Logos
- `logos/horizontal_logo.png` - Horizontal logo for headers and sign-in pages
- `logos/app_icon.png` - App icon for navigation and compact branding

### Colors
- Primary Black: #0D0D0D
- Primary Blue: #0052FF
- White: #FFFFFF
- Gray: #6B7280
- Light Gray: #E5E7EB
- Light Background: #F8FAFC

## Usage Guidelines

### Horizontal Logo
- Sign in page
- Operations dashboard header
- Owner dashboard header  
- Property manager dashboard header

### App Icon
- Resident dashboard navigation
- Worker dashboard navigation
- Compact app/header branding

### Brand Colors
- Use primary blue for CTAs and accents
- Use primary black for headers and important text
- Use light background for main app screens
- Use gray for secondary text and borders
